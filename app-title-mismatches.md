## 🛠️ How to Fix

### Update App Title

**1. Edit index.html:**
```html
<!-- Change this -->
<title>Vite + React</title>

<!-- To this -->
<title>Your App Name</title>
```

**2. Edit App.tsx/App.jsx:**
```jsx
// Change this
<h1>Welcome</h1>

// To this
<h1>Your App Name</h1>
<p>Your app description</p>
```

**3. Rebuild and recapture:**
```bash
cd [app-name]
pnpm run build
cd ..
node capture-app-screenshots-playwright.js
```
