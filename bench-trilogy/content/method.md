**THE TABLE technique** (Plate 04, Mirror): Character continuity system: African couple = primary, white couple = universal mirror. Same picnic props across both plates. This is how you globalize a local story without losing soul.

1. **Style Transfer:** Take same bench asset, switch renderer from ARRIFLEX photoreal to watercolor ink wash. Keep focal length (35mm) and marigold hex identical — audience feels return, not told.

## Playbook

### 7 steps. Any bench. Any world.

1. **Pick Anchor Prop** — Choose one small, emotional, Ghanaian object. Bench, stool, bowl, cloth. Photograph at dawn. *(Example: Bench + marigolds)*
2. **Build Spectacle Pass** — Put anchor in Times Square / Tokyo / Space. Double-exposure. Photoreal 8K. *(Labadi × Times Square glow)*
3. **Engineer VFX Kit** — LOG, tracking markers, depth, alpha, wet/dry, verticals. Zip it. *(7 plates — deliverable)*
4. **Write Intimate Sequel** — Same prop, watercolor. Add couple + puzzle mechanic as love metaphor. *(FOUND/UNFOUND hex)*
5. **Productize** — Turn mechanic into Made in Ghana object. Odum wood, brass, kente. *(Puzzle → physical)*
6. **Generate Lifestyle** — Workshop, cafe, packaging — all on same bench. Light locked. *(Suame + Accra cafe plates)*
7. **Package as Bible** — Director's Cut Bible = sales deck + crew manual + masterclass. *(This document)*

**How to use:** Run this 7 times. First run = bench. Second run = kente cloth. Third run = groundnut bowl. By run 3 you have a catalog. By run 7 you have a brand house. The bench is not furniture. It's a business model.
