💡 Typography Best Practices

  • Hierarchy is Key: Use different font weights to create clear information hierarchy
  • Spacing Matters: Don't forget about letter-spacing for headlines - it adds elegance
  • Line Height: 1.6 line-height ensures comfortable reading for body text
  • Consistency: Stick to the defined sizes and weights across your entire project
  • Responsive Design: Use relative units (rem, em) for better scalability

Logo Guidelines

🏛️ Official TUC Logo

Download the official logo: Always use the official PNG file from the TUC website for consistency and quality.

✅ Logo Do's

  • Use minimum 60px width for digital applications
  • Maintain 20px clear space around logo on all sides
  • Place on white or cream backgrounds for optimal visibility
  • Use circular white container when placing on dark backgrounds
  • Maintain original proportions at all times
  • Use PNG format for web applications
  • Ensure logo is readable at all sizes

❌ Logo Don'ts

  • Don't stretch, distort, or skew the logo
  • Don't change the colors or add effects
  • Don't place on busy or low-contrast backgrounds
  • Don't rotate or flip the logo
  • Don't use low-resolution versions
  • Don't add drop shadows or outlines
  • Don't place other elements too close to the logo

📏 Logo Specifications

Usage Minimum Size Clear Space Format

📊 Data Visualization

Guidelines for charts, graphs, and data representation.

  • Primary Data: Use Burgundy (#8B1538) for main data series
  • Secondary Data: Use Gold (#D4AF37) for comparison or accent data
  • Neutral Data: Use Warm Beige (#E6D5C7) for background or inactive states
  • Grid Lines: Use light gray (#F4E4BC) for subtle grid lines
  • Text: Maintain high contrast with Burgundy text on light backgrounds
