AUCDT Scripts Dictionary

A comprehensive guide to the automation, patching, and generation scripts powering the monorepo architecture.

🚀 Generators

  • generate_enhanced_react_apps.py

    Master Python script for scaffolding new React applications with TUC styling standards.

  • generate-ultimate-blueprint.js

    Compiles entire projects into monolithic AGENT.md self-replicating blueprints.

  • generate_all_srs.py

    Generates detailed Software Requirements Specification (SRS) markdown for all apps.

  • generate-gallery.js

    Builds the visual HTML gallery for the master catalog using captured screenshots.

⚙️ Phase Execution (Pipelines)

  • phase1-react-pin.js

    Phase 1: Pins all apps to React 19.2.5 and standardizes package.json.

  • phase2-auth-admin.js

    Phase 2: Injects authentication gates, admin dashboards, and MARKAI.md specs.

  • phase3-test-scaffold.js

    Phase 3: Scaffolds Playwright/Vitest E2E testing environments for validation.

  • phase4-docs-gen.js

    Phase 4: Generates deployment documentation and alignment artifacts.

  • phase5-srs-sync.js

    Phase 5: Synchronizes project state with Gap Analysis and finishes delivery.

🩹 Patchers & Standardizers

  • fix-tailwind-v4.js

    Migrates older Tailwind CSS v3 configurations to the new Vite v4 implementation.

  • patch-auth-gates.js

    Fixes infinite redirect loops and missing auth provider wrappers across apps.

  • standardize.js

    Runs cross-repository linting and structural standardization checks.

  • patch-favicons.mjs

    Injects standard TUC branding favicons and meta tags into index.html files.

📸 Testing & Visual Validation

  • screenshot-all.mjs

    Uses Puppeteer to capture E2E visual evidence of running applications.

  • fast-html-validator.js

    Performs high-speed HTML semantic validation against W3C standards.

  • agentic-verification-test.ts

    Runs automated agentic validation tests on deployed environments.

🐳 Deployment & Docker

  • generate-dockerfiles.sh

    Scaffolds standardized Nginx Alpine Dockerfiles for all frontend projects.

  • docker-batch-start.sh

    Utility to execute batch docker-compose up operations.

  • verify-deployments.ts

    Checks the status of live applications on Plesk/Nginx servers (ai-tools.techbridge).

🧰 Misc Utilities

  • extract-blueprint.js

    Extracts an application from a monolithic AGENT.md self-replicating file.

  • install-failed-apps.js

    Reruns package installations for applications that failed due to dependency conflicts.

  • root-reorganise.js

    Maintains directory structural integrity at the workspace level.
