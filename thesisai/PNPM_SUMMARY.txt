## Installation

One-time setup:

   $ npm install -g pnpm
   
   or
   
   $ corepack enable
   $ corepack prepare pnpm@8.15.0 --activate

Then verify:

   $ pnpm --version
   8.15.0

## Quick Start

   1. Install pnpm globally (one-time)
   2. cd frontend
   3. pnpm install
   4. pnpm dev

That's it! Everything else works the same.
