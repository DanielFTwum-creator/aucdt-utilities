# Administrator Guide
## Drumming for SEL Success Portal

### 1. Accessing the Portal
- **URL**: Navigate to `/admin/login`
- **Credentials**: Use the system administrator password (default: `admin123`).
- **Security**: All login attempts, successes, and failures are recorded in the Audit Logs.

### 2. Dashboard Overview
The Admin Dashboard is divided into four primary sections:
- **Overview**: Real-time statistics on registrations, revenue, and active inquiries.
- **Audit Logs**: A chronological record of all administrative actions.
- **Diagnostics**: System health metrics, including API connectivity and security status.
- **Testing**: Interface for running automated system integrity tests.

### 3. Managing Content
- **Inquiries**: View and manage school residency requests.
- **Seminars**: Update upcoming training dates and locations.
- **Certificates**: Issue and verify professional development credentials.
- **Blog**: Publish and edit news articles or resources.

### 4. System Settings
- **Theming**: Use the theme toggle in the sidebar to switch between Light, Dark, and System modes.
- **Logout**: Always sign out when finished to protect administrative access.

*Note: This application requires React 19.2.4 for full compatibility.*
