# AUCDT Replica Documentation

Welcome to the documentation hub for the AUCDT Website Replica project.

## Directory Structure

*   **`SRS_AUCDT_Replica_Final.md`**: The master Software Requirements Specification document.
*   **`AdministratorGuide.md`**: Manual for system administrators (Login, Logs, Security).
*   **`DeploymentGuide.md`**: Instructions for building and deploying to production.
*   **`TestingGuide.md`**: Guide for running Manual and Automated tests.
*   **`svg/`**: Contains high-fidelity technical diagrams.
*   **`presentation/`**: Contains simplified diagrams for stakeholder presentations.

## Quick Start

1.  **Install Dependencies**: `npm install`
2.  **Run Dev Server**: `npm start`
3.  **Run Playwright Tests**: `node tests/playwright_suite.js`
