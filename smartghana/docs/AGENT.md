The interface follows a strict editorial-grade redesign protocol:

### R1 — Rethink (Hierarchy)
- **Primary Action:** The "Apply Now" CTA must be singular and persistent.
- **Visual Rhythm:** Use asymmetric split-panel heroes (65/35 ratio) to avoid robotic symmetry.

### R2 — Refuse (Pattern Rejection)
- **REFUSE:** Generic Inter-only systems. Use High-Contrast Serif/Sans pairings.

### R4 — Reuse (Equity)

## 4. Key Components & Technical Implementation

### Video Multiplier (High-Fidelity)
Located in the "Efficiency Multiplier" zone.
- **Source:** `https://techbridge.edu.gh/static/campus_tour.mp4`
- **Implementation:** 
    - `video` element with `ref={videoRef}` for programmatic `autoplay`.
    - `playsInline`, `muted`, `loop` attributes enabled.
    - Interactive overlay: `white/5` opacity layer + `espresso` to `transparent` gradient bottom-mask.
    - Hover behavior: Transitions from `grayscale opacity-80` to `color full-opacity` over 1000ms.

### Economic Visualization (Recharts)
- **Growth Projections:** `AreaChart` with `fill="url(#colorGva)"` using a customized gradient from gold to transparent.
- **Sector Analysis:** `BarChart` with alternating custom cells (Espresso and Gold).

### The Sovereign Logo/Watermark
- **Logo:** `https://techbridge.edu.gh/static/TUC_LOGO_1.png`
- **Watermark Component:**
    - Absolute positioned, `inset-0`, `pointer-events-none`.
    - `grayscale(100)`, `opacity(0.02)`.
    - Acts as a "subliminal brand anchor" behind the main scroll.
