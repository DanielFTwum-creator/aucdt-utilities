TUC Examination Portal - Project Unpacker

Follow the steps below to create your project files easily.

Instructions

  1. On your computer, create a main project folder named aucdt-exam-portal.
  2. Inside it, create two more folders: aucdt_api and aucdt-client.
  3. For each file listed below:
    • Create a new, empty file with the exact name and in the correct folder.
    • Click the "Copy to Clipboard" button next to the code.
    • Paste the code into your empty file and save it.
  4. Once all files are created, follow the setup instructions in the README.md file.
